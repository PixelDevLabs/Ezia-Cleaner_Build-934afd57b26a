Multiplayer

Set Your Options

Before starting a multiplayer game, all players must set their multiplayer options by selecting OPTIONS and then MULTIPLAYER:

  • Enter a name for your character.

  • Select a color for your character's clothes, which helps differentiate him from the others.

  • Set the protocol.  All players must use the same protocol.  If you are playing over the Internet, select TCP/IP.  If you are playing on a Local Area Network ask your network administrator which protocol to use. In general, if it's an older Novell network you'd probably use IPX, and otherwise you'd use TCP/IP.  (The Mac OS version does not support IPX.)

  • Set the Connection to whatever type of network connection you're using (or find the connection that is closest in speed to your connection).

Host or Join

In a multiplayer game, one of the players must "host" a game and then all the other players will "join" that game.  Pick one of the following for more information:
