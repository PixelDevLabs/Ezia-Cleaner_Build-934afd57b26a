Display Settings

If you are not using DirectX and don't have Windows NT then you have to make sure that your Display Settings are set to 256 colors before starting Postal Plus.

If you need to check or change your display settings, follow these steps:

  • Click the Windows Start button, then click Settings, then click Control Panel.
  • In the window that appears, double-click Display.
  • In the window that appears, click the Settings tab.
  • In the Color Palette or Colors box, click 256 Colors.
  • Click OK.

If you are prompted to restart the computer, do so.

Note: We strongly recommend the use of DirectX which eliminates the need for you to manually change your display settings. To learn more about DirectX, click here.
