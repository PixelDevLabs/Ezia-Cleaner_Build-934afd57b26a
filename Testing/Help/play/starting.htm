Going Postal

To get going, just insert the Postal Plus CD and it will automatically start.

Just in case, here's the complete checklist:

  • If you did NOT already install Postal Plus, you must do so before you can play. Click here for installation information.

  • If you are NOT using DirectX or Windows NT, then make sure your display settings are set to 256 colors before you start the game.  Click here to find out how.

  • If the Microsoft Office Toolbar is running, exit it before starting the game. Otherwise, it will prevent the game from working properly.

  • Insert the Postal Plus CD and it will start automatically.

    • If the AutoRun feature is disabled on your computer then nothing will happen after you insert the CD.  In this case, use the next method.

  • Another way to start the game is click the Windows Start button, point to Programs, point to Postal Plus, then click Postal Plus.

    If you changed the default program group during Setup, then Postal Plus will appear in a different location on your Start menu.
