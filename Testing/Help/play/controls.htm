Controls

The tables below show the initial keyboard, mouse and joystick settings. Many of these settings can be changed by the user, in which case this information will no longer be accurate.

The most commonly control scheme involves using the mouse and keyboard together. The mouse is used to quickly and precisely rotate the Dude (by moving the mouse left and right) while the keyboard is used to move him forward and backward (using the up and down arrow keys) and to control his speed (using the shift key). This combination takes some getting used to as it requires both hands, one on the keyboard and another on the mouse, but the payoff is worth it. This scheme is also used by many other shooter-style games.

For more information about changing your keyboard, mouse or joystick settings, click here.

 

Analog/Force Feedback Devices

Force-feedback devices are not supported. We do not recommend the use of an analog-style joystick, as it does not work well with this style of game play. A digital-style joystick is much more well-suited.

 

Postal Dude Mouse Controls

move mouse left Turn left (or move left when strafing)
move mouse right Turn right (or move right when strafing)
right button Run forward
left button Fire weapon
middle button (optional) Hold down to strafe (use with left/right mouse movement)

 

Postal Dude Keys

left arrow Turn left (or move left when strafing)
right arrow Turn right (or move right when strafing)
up arrow Run forward
down arrow Run backward
shift Hold down for faster running and faster rotation
Alt Hold down to strafe (use in conjunction with left and right arrow keys or left and right mouse movement)
X Execute a moaning or writhing victim (be sure to position the Postal Dude close to the victim before pressing this)
NumPad 0 Duck or dodge missiles
Q Commit suicide

 

Weapon Keys

Ctrl Press and release to fire weapon once or
hold down for rapid / continuous firing
1 Select machine gun
2 Select shotgun
3 Select spray cannon
4 Select grenade
5 Select missile
6 Select heat-seeking missile
7 Select molotov cocktail
8 Select napalm
9 Select flamethrower
0 Select mine (press again to cycle through various mines)
' (quote) No weapon

 

General Keys

Esc Pause the game and display the game menu
Pause Pause the game
space Restart the level if the Postal Dude is dead (brings Postal Dude back to life in multiplayer mode)
F1 Go to the next level (only if all the hostiles have been killed)
F2 Toggle crosshairs on and off
F3 Toggle x-ray mode on and off (allows you to see the Postal Dude through any object)
F4 Toggles diagnostic display on and off
F5 Display current mission goal
- Decrease visible gameplay area for improved performance
+ Increase the visible gameplay area (see above)
T Enter chat text to send to other players in a multiplayer game.  (press Enter to send your text or Esc to abort it)
Enter If TakeSnapShots is enabled in your .ini file, takes a snap shot and saves it to the installation folder.
