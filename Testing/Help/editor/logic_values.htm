Logic File Values

If you are truly sick and twisted, you can even create your own logic files.

Here, we provide you with all the possible values for each line, but it would be too much to go into the specifics of what each line causes the person to do.  So, you are on your own to experiment with the given logic files.

You can edit logic files in any text editor (i.e. Notepad).  Just be aware that the differences in how tabs are displayed may cause the contents of the file to look funky.
