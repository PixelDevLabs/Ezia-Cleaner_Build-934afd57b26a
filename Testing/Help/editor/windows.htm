Editor Windows

Once you have a level open, you'll see several windows that give you information about it and let you work with it.

Tip: If you video card supports higher resolutions, you can press the '+' and '-' keys to increase and decrease your screen resolution while using the editor. Working at a higher resolution lets you see more of your level at once, which can help greatly when placing and editing objects.

The Realm window gives you the typical options such as New, Open, Save, Save As, Close, Play and Exit. It also has the '...' button, which brings up these Scoring & Play Options:

Standard Default choice for standard, single player Postal levels
Timed Limits level by a stated amount of time. Time counts down.
Timed Goal Tracks amount of time to goal.  Time counts down.
Timed Flag Certain number of flags in a stated amount of time. Time counts down.
Goal No time limit, but counts up to keep track of time.
Capture Flag Get flag and bring to the appropriate base.
Checkpoint Collect flags.  Flags can be set to give bonus time.  Has a time limit.

You can enter a value or values for Time, Number of Kills, Percentage of Kills, or Number of Flags in the appropriate boxes. The default choice is % Kills equals 80%.

The multiplayer options (those starting with MP) are set when starting a multiplayer level and not from within the editor.  You should just ignore those choices.

 

The View Attributes window allows you to view the attributes set for the background you are currently using. By checking a box,  that attribute will be marked with a light green color. Layers allows you to see where the walls of buildings are. Height shows where the ground is raised. No walk shows where enemies and the Postal dude will not be able to walk. The Light box shows where light effects have been added to a level.

TIP: Use the View Attributes to make sure you are not placing enemies where they might get stuck.

 

The Info for * window shows you the X, Y, and Z positions of any object you select.

 

The Map window allows you to see the entire level on a smaller scale. It also functions as a way to move quickly around the level, by clicking what you want to see on the map. The map can be refreshed by pushing the Refresh button.

 

The Layers window turns the selected layers on and off. By default, all layers are on.  Layers include the Background, Sprite, Opaque, and Alpha. This can be useful for placing bouy nets behind buildings, etc.

 

The Cameras window allows you to create a stationary view of a location on the background, while still being able to move your view around the map.  Click New to create a new camera.  Click Close to close the selected camera.

 

The NavNets window lets you select the current NavNet. This will be explained in more detail later.
