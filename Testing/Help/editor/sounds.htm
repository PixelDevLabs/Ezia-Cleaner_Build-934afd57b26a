Sounds Files

This list contains all the sound files used in the original Postal.

They are grouped into catagories, and each sound has a vague description to give you some clue as to how it is used. (Note: The descriptions are lifted directly from the programming source code, which is why they are so terse and why the words run together.)

To use these sounds in a SoundThing, you must type the name exactly as it is listed here, included the path if one is given.
