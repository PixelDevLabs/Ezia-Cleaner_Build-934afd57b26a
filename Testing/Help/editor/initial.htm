Initial Objects

This section describes the initial objects you should add to a new level:

  • Placing a Warp
  • Navnets
  • Placing Bouys

 

Placing a Warp

The first thing you should do when creating a level is to add a warp.   This is where the player will start the level.  Look in the "Place object..." box.  When you select Warp, your cursor will turn into that object.  Then, click on the background where you want to paste it in.  If you're creating a multiplayer level, add several warps, otherwise everyone will start right next to each other.  You can also do this if you want random start points in single player mode.  Once you have placed a warp, you can move it by clicking and holding on the warp and then dragging it around.  Double click on the warp to set its StockPile.   This is what weapons, ammo, etc. you'll enter the world with when you play.   Editing any warp re-sets all the other warps on the same level to the new settings.

NavNets

NavNets are used by the population of a level to navigate around obstacles and to figure out where the Postal Dude is. NavNets are made up of Bouys.  Bouys are then linked together to create a navigational path.  The enemies can clumsily do this to some extent even without a bouy network, but they really need the help.  The first NavNet (the round red stamp that appears in the upper left corner of the world) is created by default.

If you are not going to need additional navigation, you can just begin adding bouys.   They will automatically be added to the default NavNet.  To create a new NavNet, select NavNet from the list.  Type its name in the dialog that appears and click it in anywhere on the background. (It doesn't matter where.)  When you go on to add bouys, the bouys are added to whichever NavNet is currently selected.   You can toggle between existing NavNets by clicking on them in the NavNets box in the lower right of your screen.

TIP:  When you start placing enemies, note that they are automatically linked to the currently selected NavNet and will ignore all others!   Be careful that the intended NavNet is selected before placing enemies!

We've also provided the actual NavNets and bouy networks we used to create the game levels, so that you don't have to make your own unless you want to ('cause it's kind of a pain in the ass...).  Use the Open command and look in the bouynets directory.  You can choose one from this list and go straight on to adding objects if you wish.

 

Placing Bouys

Think of bouys as beacons that let the level's population know where the edges of objects are.  They pass messages along from one to another to also let the enemies know which bouy is closest to the Postal dude.  By adding bouys, you can give the population on your level a more "natural" look.

To place bouys, first make sure you have selected the NavNet you wish these bouys to be associated with.  Then, choose Bouy from the "place objects..." list.  Move the cursor onto the background and click where you want the bouy to go.   If you do not see the bouy, hit "B" to make the bouys visible. (Pressing "B" again will hide them.)

TIP: Pressing the spacebar adds another of whatever the last selected item was. CAUTION: Holding the spacebar down will place MANY of the last selected item! Be sure to only tap the bar!

You can place as many bouys as you think necessary (up to a max of 254).  Think about how much help the enemies will need to get around tricky obstacles.

Once you have placed the necessary bouys, you must link them to one another to create the path for the enemies.  First, right click on a bouy.  A red line will appear.  Drag this line over to the bouy you want to link to and click.  Now you'll see a green line linking the bouys.  Note that you can link any bouy to as many other bouys as you want.  In fact, the more links, the more realistic and unpredictable the enemies movements will be.  However, once you have made a connection from one bouy to another, the only way to remove it is to delete one of the bouys.  If you want, you can create multiple NavNets, each with their own discrete bouy system. (This is useful if you want certain enemies to stay within a certain area.)   Any bouys you place will automatically be associated with the current NavNet.

TIP: The main use for NavNets are to insure that the enemies have clear paths through tight spaces.  To help make sure you are accomplishing this, check on the height and no walk boxes in the view attributes box.  This makes those attributes visible (as transparent green areas). Now you can see where movement will be blocked.   Try to give the enemies as simple and straight a path as possible through tight spaces.
