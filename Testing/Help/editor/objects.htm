More Objects

There are many additional types of objects that can be used to spice up your level and create new effects.  Each of these can be created by selecting them in the Objects window and clicking them in on the background.

  • Miscellaneous
  • Dispensers
  • SoundThing

TIP: When adding objects, you can use the commands Control-C for copy, Control-V for paste, and Control-X for cut.

 

Miscellaneous

Barrels can be placed through a level, which explode when they are shot.   Double clicking on them brings up a dialog where you can set them to be destructible only by the player.

There are several types of Mines which can be created.  Proximity mines detonate when someone gets within a certain distance. Timed mines go off after a set amount of time.  Bouncing betty mines go off and shoot shrapnel into the air when someone gets within close proximity..

The Sentry is an automatic gun which senses and tracks the Postal dude.   By double clicking on it, you can set how much ammo it holds (Ammo supply), how long it will wait before shooting (shot delay in milliseconds), how close the Postal dude needs to be before it can shoot (range in pixels), and how fast it can turn (rotational vol in degrees per second).

TIP:  Sentry guns cannot be killed with bullets.   (Everyone knows bullets can't make a big, metal gun explode.)  This can make them tough to kill.  To make a sentry easier to kill, try putting a barrel next to it.  When you kill the barrel, the sentry will also die.

Ostrich can make fun and easy targets.  They just run around and put their heads in the dirt.  They have no weapons, are easy to kill, and have something related to panic logic.

Band members are also fun and easy targets.  Once they are placed, double click on them to select what type of instrument they play.  Band members will use a bouy network and use panic-type logic.

Power ups can be set to contain whatever you wish, including all types of ammo, weapons, health, and kevlar vests.  You can also place the power up in a backpack.  If you have more than one type of item in a power up, it will automatically be placed in a box.  The power up will take on the look of whatever type of item it contains.  When you place a power up, it will automatically display a dialog to choose what's in it.

The Flag and Flagbase are used in many of the "challenge" missions.  The Postal dude will automatically pick up flags as he runs over them and will automatically place them in a base as he runs by one.

AnimThing places an animation on the background.  You must know the name of the .aan file to use this because there is no way to browse for them within the Editor.   Fire.aan is one example.  However, if you place a fire, it won't actually hurt anyone.

Item3d allows you to place certain other 3d items on your level.  Right now, it can place horns, saxophones, and trumpets from the Band members.  You can also set how quickly it will rotate on both the Y and the Z axis.

 

Dispensers

Dispensers are one of the most versatile items in Postal.  They spew out whatever you select, in the quantities you choose, based on the parameters you set.   When you place a dispenser, a dialog will appear which allows you to set the "dispensee" and how it will be dispensed.  If you choose something which can be edited further (like a Person), click on the Edit Dispensee... button located next to the OK button.  If you choose something which has no properties to edit (like an Ostrich), clicking on that button will close the dialog.

TIP:  You can ALT - Double click a dispenser to jump straight into editing the dispensee.

Dispensers can be set with one of three logic types.  Timed sends out the object based on an amount of time.  It allows you to set the initial delay before dispensing begins, and the minimum and maximum amount of time before the next object is dispensed.  Exists sends out an object based on when the last dispensee was killed or destroyed.  It allows you set the initial delay, and the minimum and maximum amount of time before the next object is dispensed after the previous one is destroyed.  Distance to Dude sends out an object based on how close the Postal dude gets to the dispenser.  It allows you to set the initial delay, minimum and maximum distance the Postal dude must be to dispense, and how often it will check to see if the dude is within that area.

Dispensers can also be set to dispense only a limited number of that item or they can be set to be infinite, by checking the box.  Dispensers are invisible when playing your level, but show up as the item you have chosen with a 25¢ symbol on them.

TIP:  Dispensers can be used to create all kinds of unique effects, so try experimenting with them to see what you can do.  One example is the exploding toilets on the Construction level.  These were made with a barrel dispenser set on Exist with a short enough delay so that the replacement barrel explodes in the fire of the previous barrel.

 

SoundThings

As sound is such an important part of playing Postal, the SoundThing has many options.  It enables you to add in ambient sounds like wind, birds, phone sex, etc.   After placing a SoundThing, double click on it to set its properties.  Type in the name of the sound (click here for available sounds).

TIP:  Adding the Demon object to the world creates that booming voice that taunts the enemies and congratulates you while you play.

The first set of choices allows you to set a range from when the sound will first play to when it will play again.  First Minimum Time is the minimum time until the sound will first play.  First Random Time is what creates a range to decide when the sound will play.  Postal will pick a random number between zero and   First Random Time, which it adds to First Minimum Time to decide when the sound will first play.  An example might make it more clear:  If you choose 1000 for the First Minimum Time and 2000 for the First Random Time, the first sound will play somewhere between 2 and 3 seconds.  Repeat Minimum Time and Repeat Random Time do the same thing for additional times that the sound will be played.  It was set up this way so that sounds would have a more realistic quality rather than repeating at only one interval.

Select initial status must be set at Enable in order for the SoundThing to be heard.  Setting the second field to Repeat will tell the SoundThing to use the random time.

The Sound Volume Half Life is how far the sound will "carry".   You will notice this effect with some of the street lights in Postal.  As you get closer to them, you can hear the buzz of the florescent lights get louder.  The lower the number, the closer you have to be to hear the sound.

When checked, Use looping opens several more choices.  Loop back to allows you to start your second loop at a point other than the beginning of the sound.  Loop back from allows you to stop your the loop at a point other than the end.  If you check End, the SoundThing will loop back from the end of the sound.  Number of times to loop back can be set to a finite number or to infinite.

When checked, Purge Sample when done playing will read the sound off the CD or hard drive (depending on how the player installed Postal) rather than leaving the sound in memory.

If Ambient Sound is checked, means the player can turn the sound off if they turn off Ambient Sounds in their Options - Performance  menu.  If you do not check the box, the player will not be able to turn off that sound even if they turn off Ambient Sounds for performance reasons.
