System Requirements

Postal Plus requires Windows 95, 98, NT4 or 2000.

Minimum System
    90 MHz Pentium Processor
    16 MB Memory
    95 MB Hard Drive Space
    2x CD-ROM Drive
    256 Colors at 640x480 Resolution
    8-bit Audio - Sound Blaster or 100% Compatible

Recommended System
    Pentium 166 MHz
    32 MB Memory
    300 MB Hard Drive Space
    4x CD-ROM Drive
    256 Colors at 640x480 Resolution
    16-bit Audio - Sound Blaster or 100% Compatible
