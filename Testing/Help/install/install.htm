Install Postal Plus

To install Postal Plus, do this:

  1. Insert the Postal Plus CD.

  2. After a few moments the Postal Plus window appears, asking if you want to install now.

  3. Click Yes.

  4. This starts Setup.  Follow the on-screen instructions.

What if nothing happens after I insert the CD?

If you insert the Postal Plus compact disc into your CD-ROM drive and nothing happens, then the AutoRun feature may be disabled on your computer.

In that case, follow these instructions:

  1. Insert the Postal Plus CD.
     
  2. Click the Windows Start button, then click Run...
     
  3. Type the following, except replace 'D' with the letter of your CD-ROM drive:
     
        D:\setup
     
    and then click OK.
     
  4. This starts Setup.   Follow the on-screen instructions.

Should I choose the Complete Game or the Multiplayer-Only Game?

The Complete Game is pretty much what it says: it gives you absolutely everything.  This is more than likely the choice you want to make.

The Multiplayer-Only Game is a limited version of the game that you can install on someone else's computer so they can join you in a multiplayer game.  As its name implies, it only supports multiplayer games.

So let's say you want to play against someone else at your office but you've only got one copy of Postal Plus.  No problem!  Just install the Multiplayer-Only Game on that person's computer and then install the Complete Game on your computer.  Then you can start blowing each other away.  (Note: you will need to keep the Postal Plus CD inserted in your computer while playing.)

Which install options should I choose?

None of these options has any effect on the content of the game -- it's the same no matter what you choose.  What they do effect is how quickly the game loads.

So it all comes down to speed versus hard drive space.

The more options you select, the more files are copied to your hard drive.  The upside is that when you're playing the game, it will load much more quickly, especially when you go from one level to the next.

In the Setup program, when you click on the various options it tells you how much space that option will take up on your hard drive and how much of an impact it will have when playing the game.

So it's really up to you.  If you hate waiting and have enough space, then select most or all of the options.  But if you're the calm, patient type, just go for the minimum.

Which audio quality should I choose?

No matter which quality you choose, you'll always hear the same sounds, voices and music in the game.  The difference is purely in the quality.

The Normal Quality option.  Most people are quite happy hearing the screams, destruction and general mayhem using Normal Quality, which is pretty much why we called it the "Normal" setting.

The High Quality option.  This is for audio fanatics who want the extra clarity of a higher sampling rate.  Of course, this quality comes at a price -- it doubles the memory and hard drive space requirements as compared to Normal Quality.  This is only recommended for those with enough hard drive space and at least 64 MB of memory.

Note:  For those few that have an "8 bit" sound card, the only option available will be to install audio (no quality choice).

Which location or directory should I install to?

You should accept the suggested location.

You would only change the location if you had a really good reason, and in that case you probably wouldn't have asked this question. WARNING: You should never install more than one version of Postal to the same location! For instance, if you already have the original Postal installed, do NOT install this new version to the same location! Doing so would cause problems with other version.
