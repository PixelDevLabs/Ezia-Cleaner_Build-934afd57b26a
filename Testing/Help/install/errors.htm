Error Codes

In many cases, the error messages displayed by Postal Plus refer to a code number.

Click on the code number below for more information about that particular error.

Each code is explained in detail below.

 

R100

This is a rare error that indicates a serious incompatibility between the game and your computer.

In some cases, updating your video and/or audio drivers may resolve this problem.

If that does not resolve the problem, contact your computer vendor.

(return to top)

 

R101

It appears that your video card supports 640x480 resolution with 256 Colors but something is preventing the game from setting that mode.

This problem can often be resolved by updating your video drivers.

Video card vendors often make the latest versions of their drivers available via their web sites. Contact your video card vendor for more information.

(return to top)

 

R102

The game requires that your display be set to 256 Colors, which is different from your current display settings.

You must manually change your display settings before running the game. To learn how to do so, click here.

Note: We strongly recommend the use of DirectX, which eliminates the need for you to manually change your display settings. To learn more about DirectX, click here.

(return to top)

 

R103

Your video card does not appear to be capable of displaying 256 colors.

This generally indicates a hardware limitation of your video card, although it is possible that an upgraded video driver could resolve the problem.

Check your video card documentation (or contact the vendor) to determine whether the card supports 256 Colors at 640x480 resolution. If so, a new driver may correct the problem. If not, then the only way to resolve it is to upgrade your video card.

(return to top)

 

R104

Your video card does not appear to be capable of 640x480 resolution while using 256 colors. In other words, it appears to be capable of 256 colors, but not at the required resolution.

This generally indicates a hardware limitation of your video card, although it is possible that an upgraded video driver could resolve the problem.

Check your video card documentation (or contact the vendor) to determine whether the card supports 256 Colors at 640x480 resolution. If so, a new driver may correct the problem. If not, then the only way to resolve it is to upgrade your video card.

(return to top)

 

R105

Postal Plus should not normally display this error! This would indicate a very odd incompatibility with your video card and/or driver.

Your video card does not appear to be capable of supporting the required number of pages.

This generally indicates a hardware limitation of your video card, although it is possible that an upgraded video driver could resolve the problem.

Contact your video card vendor for more information.

(return to top)

 

R106

You must manually change your display settings.

You should install DirectX so the game can automatically change your settings as required and automatically restore them when it is done.

You have two options. Click on either one for details:

    1. Manually change your display settings

    2. Install DirectX

(return to top)

 

A100

The file "Postal Plus.ini" could not be found.

This generally means the file was accidentally deleted or moved.

Unfortunately, the game cannot run without this file, as it contains important configuration information.

You can try finding the file on your hard drive. Click Start then Find then Files or Folders and enter "Postal Plus.ini" and click OK. If the file is found, make sure to move it back to the folder where Postal Plus was installed.

If you cannot find the file, you will have to reinstall Postal Plus.

(return to top)

 

A101

An error occurred while reading from the file "Postal Plus.ini".

This generally indicates that the file exists but may be damaged.

You will need to reinstall Postal Plus.

(return to top)

 

A102

An error occurred while writing to the file "Postal Plus.ini".

This generally indicates that the file exists but the game is being prevented from writing to it. This could happen if another application currently has the file open, or if you do not have permission to write to the file.

You can still play the game, but any changes you make (such as key or button mappings) will not be saved.

(return to top)

 

A103

The file "Postal Plus.ini" has been set to "Read Only" status. This prevents the game from saving any changes (such as key or button mappings) you might have made.

Generally, this will only happen if someone manually (and intentionally) changes the file's "Read Only" flag.

To clear this flag, right-click on the file Postal Plus.ini, click Properties, then clear the Read Only check mark and click OK.

(return to top)

 

A104

An error occurred during the title sequence.

This generally indicates that certain required files could not be found. Most likely, something either went wrong during installation or files were deleted after the game was installed.

You will need to reinstall Postal Plus.

(return to top)

 

A105

One or more required files could not be found.

The usual cause for this is that Postal Plus was moved from where it was originally installed. If this is the case, follow these steps to fix the problem:

    1. Open the folder where Postal Plus is currently located.

    2. You'll see two files named Postal Plus.  One is them is a text file, which usually means it has a "notepad" icon.  Double-click that file to open it in your text editor.

    3. Look for the line that reads:
       
                  [Paths]

    4. Below that line will be several lines that indicate the locations of various required files. For example, one of the lines might look like this:

                  CD = C:\Program Files\Postal Plus

    5. For EACH line that refers to a folder on your hard drive, change it to refer to the NEW folder where Postal Plus is located.

    6. Save the file and exit the text editor.

Now run Postal Plus again.

(return to top)

 

A106

The Postal Plus CD must be in your CD-ROM drive to run the game.

The first thing to check is whether the Postal Plus CD was actually in your drive. If not, put it in and try again.

If the Postal Plus CD was in the drive, then perhaps the letter of your CD-ROM drive has changed since you installed Postal Plus. This could happen if you installed or removed any hard drives, CD-ROM drives or other similar hardware. If this is the case, do the following:

  • Open the folder where Postal Plus is currently located.

  • You'll see two files named Postal Plus.  One is them is a text file, which usually means it has a "notepad" icon.  Double-click that file to open it in your text editor.

  • Look for the line that reads:
     
                CD = D:\

    where D was the letter of your CD-ROM drive.

  • Replace the D with the current letter of your CD-ROM drive.

  • Save the file and exit the text editor.

This should solve the problem.

(return to top)

 

A107

An error has occurred and the game cannot proceed.

This is a rare error type, and it generally indicates a problem with the current installation.

You should try to reinstall Postal Plus.

(return to top)

 

A108

The specified entry in your Postal Plus configuration file is invalid.

The specified entry is used by the game to find various required files. The entry must be corrected in order to play.

This generally happens when Postal Plus is moved from the location where it was installed. If that's the case, follow these steps:

  • Open the folder where Postal Plus is currently located.

  • You'll see two files named Postal Plus.  One is them is a text file, which usually means it has a "notepad" icon.  Double-click that file to open it in your text editor.

  • Look for the line that reads:
     
                [Paths]

  • Below that line, look for the line that starts with the whatever entry was specified in the error message. For instance, if the error had specified 'CD' as the invalid entry, you would look for the line that starts with:

                CD =

  • To the right of the '=' will be the location where Postal Plus was installed. Change this to refer to the current location of Postal Plus.

  • Save the file and exit the text editor.

(return to top)

 

A109

Your audio card supports a given sound format but the file associated with that format was not found.

The most common cause for this is that someone manually changed the sound quality settings in the Postal Plus.ini file. However, that's not the only step required to change the sound quality. It is also necessary to install the proper sound files from the Postal Plus CD.

If you want to change your sound quality, you should run Setup again.

Caution: Do not proceed with this unless you are experienced (or think you are). You can manually copy the necessary files from the CD to your drive. The files of interest are located under \res\game and are named according to the sample rate and the bits-per-sample (for example, 11025_16.sak). Simply copy the file you want to same location in the Postal Plus folder.

(return to top)

 

A110

There is no sound file installed, which means something went wrong during installation.

This generally should not happen, as the installer will always attempt to install at least one of the available sound qualities.

You should reinstall Postal Plus.

(return to top)
