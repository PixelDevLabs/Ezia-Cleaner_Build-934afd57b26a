Display Problems

Drivers

The most common problems are slow frame rates or jerky movements in the game, as well as glitches, garbage or tears appearing on the game screen.

We have found that the vast majority of display problems are solved by upgrading to the latest video driver for your video card.

In many cases, we have seen game performance increase by 200% when a new driver was installed! This benefits not only Postal Plus but all your computer applications.

Please go to your video card manufacturer's web site to find the latest driver, then download and install it.  Remember that if you are using DirectX, you should make sure to get the latest DirectX-compatible drivers.

Disabling DirectX

If you are using DirectX and you upgraded your video drivers and you have the latest version of DirectX and you STILL have display problems, then you could try configuring Postal Plus so it does NOT use DirectX.  This will occasionally fix a stubborn problem.

To do so, follow these steps:

  1. Open the folder where Postal Plus was installed.
    (the default installation folder is \Program Files\Postal Plus)

  2. You'll see two files named Postal Plus.  One is them is a text file, which usually means it has a "notepad" icon.  Double-click that file to open it in your text editor.

  3. Look for the line that reads:
     
                Type = DirectX

  4. Change that line to:
     
                Type = GDI

  5. Save the file and exit the text editor.

Now run Postal Plus again.

If this does NOT solve the problem, be sure to change the line back to its original form.
